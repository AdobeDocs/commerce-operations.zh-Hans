# GRA定义

- **全球参考体系结构(GRA)**：也称为白色标签架构或通用代码库。 这是用于多实例设置的模块分发体系结构。
- **多实例设置**：同一客户端为不同的地区或品牌使用不同的Adobe Commerce安装。 每个安装都具有共享模块以及独特的模块。
- **单实例设置**：只有一个Adobe Commerce安装。 对于不同的测试环境，可以存在源代码的多个副本，但生产代码只有一个版本。
