# 服务器登录

1. 登录到您的服务器。
1. 切换到 [文件系统所有者](https://devdocs.magento.com/guides/v2.4/install-gde/prereq/file-sys-perms-over.html).
1. 更改为您克隆应用程序的目录。 例如，

   ```bash
   cd /var/www/magento2
   ```
