# .gitignore引用

Magento Open Source包括基 `.gitignore` 文件。 请参阅 [最新商务 `.gitignore`](https://raw.githubusercontent.com/magento/magento2/2.4/.gitignore) 文件。 如果必须在 `.gitignore` 列表，您可以使用 `-f` （强制）选项。

```bash
git add <path/filename> -f
```
