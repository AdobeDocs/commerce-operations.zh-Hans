# 项目管理框架

DACI模型是一个项目管理框架，用于明确界定项目各利益相关方的角色和责任，从而做出有效和高效率的群组决策。 DACI代表“驱动程序”、“审批者”、“参与者”和“已告知”。 这些角色清楚地表明了谁在特定领域和情况下拥有权力，从而使项目顺利进行。
