# 项目管理框架

DACI模型是一个项目管理框架，用于明确各利益相关方在项目中的角色和责任，从而产生有效和高效的小组决策。 DACI代表“驱动程序”、“审批者”、“参与者”和“已知”。 这些角色清楚表明谁在某些领域和情况中拥有权威，从而使项目得以顺利推进。
