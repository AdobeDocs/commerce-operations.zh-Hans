# Adobe Commerce 2.4.5-p2发行说明

Adobe Commerce 2.4.5-p2是一个安全版本，它提供了三个安全修复，可增强您的Adobe Commerce 2.4.5或Magento Open Source2.4.5部署。 它为以前版本中已识别的漏洞提供了修复。

此安全修补程序包含三个安全错误修复。
