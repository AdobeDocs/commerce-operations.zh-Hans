# Adobe Commerce 2.4.5-p2发行说明

Adobe Commerce 2.4.5-p2是一个安全版本，提供了三个安全修复，以增强您的Adobe Commerce 2.4.5或Magento Open Source2.4.5部署。 它修复了以前版本中发现的漏洞。

{{bics}}

## 此版本中的内容是什么？

此安全补丁包含三个安全错误修复。

请参阅Adobe安全公告。
