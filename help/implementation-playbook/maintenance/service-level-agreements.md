# 服务级别协议(SLA)

服务级别协议(SLA)定义了服务提供商客户预期的服务级别，包括衡量该服务的简单量度，以及在达不到商定服务级别时的补救措施或处罚（如果有）。

可以收缩、维护和测量不同类型事件关键性的SLA。 此外，响应类型可以根据客户所需的服务级别，具有多个标准（金牌、白金）。
