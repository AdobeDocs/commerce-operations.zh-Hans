# 实施最佳实践

实施Adobe Commerce项目的方法多种多样。 此部分提供的最佳实践描述了Adobe期望客户和合作伙伴实施特定用例的方式。 这些建议旨在帮助防止与站点配置和自定义、部署、操作、性能和升级相关的常见问题和潜在问题。
