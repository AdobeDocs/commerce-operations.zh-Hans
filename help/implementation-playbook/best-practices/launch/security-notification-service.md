# 安全通知最佳实践

安全性是我们的最高优先事项之一，Adobe也知道安全性对客户也很重要。

此 [!DNL Adobe Security Notification Service] 是一种免费的电子邮件通知服务，Adobe使用该服务向客户发送有关Adobe产品安全性的信息。

任何人都可以订阅该服务，您随时可以取消订阅。

## 受影响的产品和版本

全部
