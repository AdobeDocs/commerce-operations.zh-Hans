# 安全通知最佳实践

安全性是我们最优先考虑的事项之一，而Adobe也明白安全对我们的客户也很重要。

的 [!DNL Adobe Security Notification Service] 是一项免费的电子邮件通知服务，Adobe可使用该服务向客户发送有关Adobe产品安全性的信息。

任何人都可以订阅该服务，您也可以随时取消订阅。
