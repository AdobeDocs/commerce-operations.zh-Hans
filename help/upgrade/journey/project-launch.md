# 项目启动

维护一个健康的电子商务网站对于您的业务成功与首次启动一样重要，其中的一个关键方面是主动规划和准备升级，帮助您的网站尽可能顺畅地运行。

在此期间未设置正确的升级预期会导致未来不满意。 确保您了解并了解 [释放节奏](https://devdocs.magento.com/release/)，并使用本指南来帮助您定义如何以及何时计划升级。
