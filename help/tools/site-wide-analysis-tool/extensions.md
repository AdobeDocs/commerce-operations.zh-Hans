# [!DNL Extensions]

安 [!DNL extension] 是自定义模块、语言包，或自定义模块和语言包的组合，可为核心Adobe Commerce产品添加新功能。 [Commerce Marketplace](https://marketplace.magento.com/extensions.html) 提供了 [!DNL extensions] 可免费购买和可出售。 您可以找到 [!DNL extensions] 安装在Adobe Commerce实例上 [!DNL Site-Wide Analysis Tool].

## 何时使用

的 [!DNL Extensions] 选项卡 [!DNL Site-Wide Analysis Tool] 提供有关 [!DNL extensions] 安装在Adobe Commerce实例上的ID。 您可以使用此选项卡检查 [!DNL extensions] 您正在使用、其版本以及其他详细信息，如描述、功能和用法。

## 优点

* 通过快速概述所有 [!DNL extensions] 安装在Adobe Commerce实例上。

* 获取有关已安装版本的信息 [!DNL extensions] 并将其更新到最新版本。

## 最佳实践

* 保留您的第三方 [!DNL extensions] 最新版本。

* 确保 [!DNL extensions] 您选择的与Adobe Commerce实例兼容。

* 如果您无法保留第三方 [!DNL extensions] 最新版本，请考虑使用不同的 [!DNL extensions].

* 要避免出现潜在问题，请下载/购买第三方 [!DNL extensions] 来自可靠来源，如 [Commerce Marketplace](https://marketplace.magento.com/extensions.html).
