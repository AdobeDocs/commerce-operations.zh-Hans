# ACSD-68356： [!DNL GraphQL]购物车查询为虚拟报价返回不正确的折扣金额

ACSD-68356修补程序修复了[!DNL GraphQL]购物车查询为虚拟报价返回错误折扣金额的问题。 安装[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.73时，此修补程序可用。 修补程序ID为ACSD-68356。 请注意，此问题计划在Adobe Commerce 2.4.9中修复。

## 受影响的产品和版本

**为Adobe Commerce版本创建了修补程序：**

* Adobe Commerce（所有部署方法） 2.4.8-p2

**与Adobe Commerce版本兼容：**

* Adobe Commerce（所有部署方法） 2.4.8 - 2.4.8-p3

>[!NOTE]
>
>该修补程序可能适用于具有新[!DNL Quality Patches Tool]发行版本的其他版本。 要检查修补程序是否与您的Adobe Commerce版本兼容，请将`magento/quality-patches`包更新到最新版本，并在[[!DNL Quality Patches Tool]：搜索修补程序页面](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html?lang=zh-Hans)上检查兼容性。 使用修补程序ID作为搜索关键字来查找修补程序。

## 问题

[!DNL GraphQL]购物车查询为虚拟报价返回不正确的折扣金额，将折扣应用于不符合条件的虚拟产品。

<u>重现步骤</u>：

1. 创建两个虚拟产品：
   1. `virtual1` SKU
   1. `virtual2` SKU
1. 创建购物车价格规则：
   1. 设置为自动应用，不带优惠券。
   1. 操作：
   1. 产品价格折扣百分比 — 20%。
   1. 仅适用于SKU为`virtual2`的购物车项目。
1. 通过店面将产品添加到新的空报价单：
   1. 首先添加`virtual1` SKU。
   1. 添加`virtual2` SKU秒。
1. 使用以下[!DNL GraphQL]查询获取购物车详细信息：

   ```
   {
     cart(cart_id: "cart_id") {
       email
       itemsV2 {
         total_count
         items {
           uid
           prices {
             total_item_discount {
               value
             }
             price {
               value
             }
             discounts {
               label
               amount {
                 value
               }
             }
           }
           product {
             name
             sku
           }
           quantity
         },
         page_info {
             page_size
             current_page
             total_pages
         }
       }
       applied_coupons {
         code
       }
       prices {
         discounts {
           amount {
             value
           }
           applied_to
           label
         }
         grand_total {
           value
         }
       }
     }
   }
   ```

1. 检查&#x200B;**[!UICONTROL cart]** > **[!UICONTROL items]** > **[!UICONTROL prices]** > **[!UICONTROL discounts]**。

<u>预期的结果</u>：

* `virtual1` SKU在[!DNL GraphQL]中没有列出折扣
* `virtual2` SKU在[!DNL GraphQL]中列出了折扣

<u>实际结果</u>：

* `virtual1`和`virtual2`在[!DNL GraphQL]响应中错误地显示相同的折扣金额。
