# ACP2E-4431：在重新索引过程中删除与目标规则匹配的[!UICONTROL Related Products]

ACP2E-4431修补程序修复了在重新索引过程中删除与目标规则匹配的&#x200B;**[!UICONTROL Related Products]**&#x200B;的问题。 安装[[!DNL Quality Patches Tool (QPT)]](/help/tools/quality-patches-tool/quality-patches-tool-to-self-serve-quality-patches.md) 1.1.78时，此修补程序可用。 修补程序ID为ACP2E-4431。 请注意，此问题计划在Adobe Commerce 2.4.9中修复。

## 受影响的产品和版本

**为Adobe Commerce版本创建了修补程序：**

* Adobe Commerce（所有部署方法） 2.4.7-p4

**与Adobe Commerce版本兼容：**

* Adobe Commerce（所有部署方法） 2.4.4 - 2.4.8-p4

>[!NOTE]
>
>该修补程序可能适用于具有新[!DNL Quality Patches Tool]发行版本的其他版本。 要检查修补程序是否与您的Adobe Commerce版本兼容，请将`magento/quality-patches`包更新到最新版本，并在[[!DNL Quality Patches Tool]：搜索修补程序页面](https://experienceleague.adobe.com/tools/commerce-quality-patches/index.html)上检查兼容性。 使用修补程序ID作为搜索关键字来查找修补程序。

## 问题

在重新索引过程中，将删除与目标规则匹配的&#x200B;**[!UICONTROL Related Products]**。

<u>重现步骤</u>：

1. 创建10000个或多个产品。
1. 将10000个或更多产品分配给5000个或更多类别。
1. 创建100个或更多相关的产品规则。
1. 配置规则条件，其中&#x200B;**[!UICONTROL Products]**&#x200B;匹配使用&#x200B;**[!UICONTROL Product Category]**&#x200B;为&#x200B;*X*。
1. 使用&#x200B;**[!UICONTROL Product Category]**&#x200B;配置&#x200B;**[!UICONTROL Products]**&#x200B;显示为&#x200B;**[!UICONTROL Constant Value]** *Y*。
1. 执行此GraphQL查询以检索产品的`related_products`：

   ```graphql
   {
       products(filter: { url_key: { eq: "simple-product-491" } }) {
           items {
               sku
               related_products {
                   uid
                   sku
                   name
                   url_key
                   stock_status
                   price_range {
                       minimum_price {
                           regular_price {
                              value
                           }
                           final_price {
                               value
                           }
                       }
                   }
               }
           }
       }
   }
   ```

1. 为`targetrule_product_rule`索引创建大型积压。
1. 为`targetrule_rule_product`索引创建大型积压。
1. 运行`targetrule_rule_product`索引器。
1. 运行`targetrule_product_rule`索引器。

<u>预期的结果</u>：

`related_products`列表始终填充有符合规则条件的产品。

<u>实际结果</u>：

当索引器正在执行时，`related_products`列表返回空数组。
