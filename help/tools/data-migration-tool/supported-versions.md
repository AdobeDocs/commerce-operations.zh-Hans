# 支持的数据迁移版本

的 [!DNL Data Migration Tool] 支持迁移 _从_ 以下版本：

* Adobe Commerce:
   * 1.11.x
   * 1.12.x
   * 1.13.x
   * 1.14.x
* Magento Open Source:
   * 1.6.x
   * 1.7.x
   * 1.8.x
   * 1.9.x

如果您从Magento Open Source迁移到Adobe Commerce，则支持以下版本：

* 1.6.x
* 1.7.x
* 1.8.x
* 1.9.x

## 迁移到版本

要了解受支持的版本（即您要迁移到的版本），请参阅 [!DNL Data Migration Tool]&#39;s [发行页面](https://github.com/magento/data-migration-tool/releases).
